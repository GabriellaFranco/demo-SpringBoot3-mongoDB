Este projeto é uma API para uma rede social simples, onde os usuários podem criar posts e interagir com eles deixando comentários.

Tecnologias utilizadas:

- MongoDB;
- Spring Boot;
- Spring Data MongoDB;
- Apache Tomcat;
- Postman;
- RESTful API.

Principais Funcionalidades:

- Insert, delete e update de usuários;
- Criação, edição e exclusão de posts.
- Listagem de posts por usuário;
- Adicionar comentários a posts;
- Querys de pesquisa;

=================================================================================================================================================

This project is an API for a simple social network where users can create posts and interact with them by leaving comments.

Technologies used:

- MongoDB;
- Spring Boot;
- Spring Data MongoDB;
- Apache Tomcat;
- Postman;
- RESTful API.

Key Features:

- Insert, delete, and update users;
- Create, edit, and delete posts;
- List posts by user;
- Add comments to posts;
- Search queries.
